# Genshin Impact Character Cards - SwiftUI Implementation

這是一個使用 SwiftUI 開發的 iOS 介面練習專案，旨在透過程式碼繪製與佈局，重現《原神》中兩位角色——艾爾海森 (Alhaitham) 與 卡維 (Kaveh) 的名片風格。

## 專案特色 (Features)

* **純 SwiftUI 繪圖**：完全不使用任何外部圖片素材，所有的圖標、標誌、背景紋理（如羽毛筆、機械翅膀、宮殿拱門）皆使用 SwiftUI 的 `Path` 與 `Shape` 繪製而成。
* **互動式配色反轉**：
    * 實作了白天/黑夜模式的切換。
    * **彩蛋功能**：每張名片旁都有獨立的切換按鈕。根據遊戲設定，這兩位角色的配色在某種模式下剛好是對方的代表色（紅 $\leftrightarrow$ 綠），本專案完美還原了這一點。
* **精細的圖層管理**：使用了 `ZStack`、`mask` 與 `overlay` 來處理複雜的圖層遮擋與紋理疊加效果（例如羽毛的細密紋理、背景的半透明幾何線稿）。

## 技術亮點 (Technical Highlights)

### 1. 複雜路徑繪製 (Custom Shapes)
專案中定義了大量的自定義 `Shape` 結構，例如：
* `AlhaithamIntricateWingShape`：使用貝茲曲線繪製複雜的機械翅膀。
* `KavehFeatherShape` & `FeatherBarbTextureShape`：結合路徑裁剪與迴圈繪製，模擬出真實羽毛的邊緣缺口與細密羽枝質感。
* `KavehDomeShape`：使用控制點精確繪製波斯風格的尖頂拱門。

### 2. 解決點擊穿透問題 (Hit Testing)
在處理卡維卡片時，遇到圖層遮擋導致按鈕失效的問題。解決方案是運用 `.zIndex(1)` 強制提升資訊區（按鈕層）的層級，使其浮於複雜的背景圖層之上。
